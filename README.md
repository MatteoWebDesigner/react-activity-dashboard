# React activity dashboard
This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app)

# Setup
Node: v8.9.3
npm: 5.5.1

```bash
yarn
```

## run DEV
```bash
npm run mock-api
npm run dev
```
open browser on http://localhost:3000

## run PROD
```bash
npm run mock-api
npm run build
npm run serve
```
open browser on http://localhost:3000
